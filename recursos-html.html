REC.HTML

HTML, hace referencia al lenguaje de marcado para la elaboración de páginas web. Es un estándar que sirve de referencia del software que conecta con la elaboración de páginas web en sus diferentes versiones, define una estructura básica y un código para la definición de contenido de una página web, como texto, imágenes, videos, juegos, entre otros. Se considera el lenguaje web más importante siendo su invención crucial en la aparición, desarrollo y expansión de la WWW. Es el estándar que se ha impuesto en la visualización de páginas web y es el que todos los navegadores actuales han adoptado.

Para documentarse mejor podemos encontrar unas cuantas páginas en las cuales hay bastante información sobre HTML. Las pagínas son: Wikipedia, CódgoFacilito, W3school
